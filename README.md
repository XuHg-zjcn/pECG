# 便携式心电监测仪

## 目标功能
- 单导联心电
- 呼吸监测
- 加速度计
- 显示心率、呼吸数据和心电波形
- 更换电池不中断数据记录

## 主要元器件
|名称           |型号(或候选)     |
|:--------------|:----------------|
|模拟前端       |ADS1292R         |
|无线MCU        |CH583M           |
|电池           |10280圆柱体锂电池|
|显示屏         |彩色TFT屏        |
